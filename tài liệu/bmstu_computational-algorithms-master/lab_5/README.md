### Условие
Есть трубка с газом, которая нагревается. Нужно узнать давление после нагревания.  
Звучить просто, да, но нет.

Одна из проверок на корректность расчетов: T0 = Tw  результат равен Pn*10 (Pn = 0.5 Tn = 300, T0 = Tw = 3000).
